# HFS Skill

HttpFileServer - 簡易架設網站與檔案傳輸

## HFS 是什麼

HFS (HttpFileServer) - 免安裝、輕量級 HTTP 檔案伺服器

- 📤 快速分享檔案
- 🌐 架設靜態網站
- 🔒 用戶權限管理
- 📱 網頁管理介面

## 支援版本

| 版本  | 語言   | GitHub                                    |
| ----- | ------ | ----------------------------------------- |
| HFS 3 | Go     | https://github.com/lanew197894fun-cmd/hfs |
| HFS 2 | Delphi | https://github.com/boss6731/hfs2          |

## 功能

- 監控 HFS 服務狀態
- 查看 GitHub 版本
- 遠程部署
- SSL 憑證支援
- 效能監控
