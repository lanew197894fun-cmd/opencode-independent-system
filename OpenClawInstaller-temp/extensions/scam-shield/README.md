# 🦞 OpenClaw Scam Shield - 瀏覽器擴展

## 功能

- ⚡ **即時防護** - 自動掃描頁面中的詐騙內容
- 🔍 **URL 檢測** - 輸入網址進行風險評估
- 🚫 **連結過濾** - 偵測可疑連結與網域
- 📝 **表單監控** - 檢測要求敏感資訊的可疑表單
- 📊 **統計記錄** - 記錄攔截與掃描次數

## 安裝方式

### Chrome / Edge / Brave

1. 打開 Chrome，輸入 `chrome://extensions`
2. 開啟右上角「開發者模式」
3. 點擊「載入未封裝項目」
4. 選擇 `extensions/scam-shield` 資料夾
5. 完成！點擊右上角圖標即可使用

### Firefox

1. 將資料夾打包為 ZIP
2. 輸入 `about:debugging#/runtime/this-firefox`
3. 點擊「臨時載入附加元件」
4. 選擇 ZIP 檔案

## 使用方式

### popup 功能
- 輸入網址點擊「開始檢測」
- 點擊「掃描當前頁面」檢測正在訪問的網頁

### 自動防護
- 安裝後自動掃描每個訪問的頁面
- 發現風險時顯示紅色警告浮層

## 風險評分說明

| 分數 | 等級 | 說明 |
|------|------|------|
| 70+ | 危險 | 高度懷疑為詐騙網站 |
| 40-69 | 警告 | 請注意，可能有風險 |
| <40 | 安全 | 初步檢查通過 |

## 技術細節

- Manifest V3
- 本地儲存統計數據
- 無需網絡權限（離線可用）
- 支援所有 Chromium 瀏覽器
