# 晨間簡報技能

## 描述
每天早晨自動推送個人化的晨間簡報，包括天氣、日程、新聞等資訊。

## 觸發條件
- 使用者說 "早安" 或 "晨間簡報" 或 "今日概覽"
- 每天早上 07:30 自動觸發

## 執行步驟
1. 取得當地天氣資訊
2. 讀取今日日程安排
3. 彙總重要新聞摘要
4. 檢查未讀訊息和提醒
5. 產生勵志語錄

## 輸出格式

```
☀️ 早安，{user_name}！

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🌤️ 今日天氣
{城市}: {天氣狀況} {溫度}
{穿衣建議}

📅 今日日程
{日程列表}

📰 新聞摘要
{3-5條重要新聞}

📬 待處理
{未讀訊息和提醒}

💭 今日寄語
"{勵志語錄}"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
祝你有美好的一天！🌟
```

## 配置選項

```yaml
skills:
  morning-briefing:
    enabled: true
    schedule: "30 7 * * *"  # 每天 07:30 觸發
    timezone: "Asia/Shanghai"
    weather:
      enabled: true
      city: "Shanghai"
      api_key: "your-weather-api-key"
    news:
      enabled: true
      sources:
        - "tech"
        - "finance"
        - "world"
      max_items: 5
    calendar:
      enabled: true
      provider: "google"  # google/outlook/ical
    channels:
      - telegram
```

## 依賴

- 天氣 API（可選）：OpenWeatherMap 或類似服務
- 日曆整合（可選）：Google Calendar / Outlook
- 新聞 API（可選）：NewsAPI 或 RSS 源
