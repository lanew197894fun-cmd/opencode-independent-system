# 詐騙訊息偵測系統

## 功能說明

自動分析簡訊、訊息內容，判斷是否為詐騙訊息，並提供風險評估與建議。

## 核心檢測引擎

### 1. 風險評分系統

| 檢測項目 | 風險分數 | 說明                             |
| -------- | -------- | -------------------------------- |
| 緊急用語 | 20-40    | "緊急處理"、"立即行動"           |
| 威脅語句 | 30-50    | "帳號鎖定"、"損失財產"           |
| 陌生連結 | 40-60    | 不明網址、短網址                 |

| 語氣強迫 | 15-30    | 情緒勒索、威脅語氣               |

```javascript
function detectSuspiciousLinks(text) {
  const linkRegex = /https?:\/\/[\w.-]+/g;
  const shortLinkRegex = /\b(?:bit\.ly|tinyurl\.com|goo\.gl)\/\w+/g;

  const links = text.match(linkRegex) || [];
  const shortLinks = text.match(shortLinkRegex) || [];

  return {
    totalLinks: links.length,
    suspiciousLinks: shortLinks.length,
    linkDomains: links.map((link) => new URL(link).hostname),
  };
}
```

### 3. 詐騙訊息類型

#### 銀行詐騙

- 特徵：帳號異常、安全威脅、要求操作
- 風險分數：60-90
- 典型訊息："您的銀行帳號有異常交易，請立即登入確認..."

#### 投資詐騙

- 特徵：高報酬保證、內線消息、投資機會
- 風險分數：70-95
- 典型訊息："內線消息，保證獲利..."

#### 網路購物詐騙

- 特徵：退款、退貨、訂單異常
- 風險分數：50-80
- 典型訊息："您的訂單有問題，請點擊連結..."

#### 親友詐騙

- 特徵：冒充親友、急需用錢、情急狀況
- 風險分數：80-100
- 典型訊息："我是你同學，我急需用錢..."

#### 政府機關詐騙

- 特徵：公務員身分、法律威脅、罰款
- 風險分數：85-100
- 典型訊息："您涉嫌違法，請配合調查..."

### 4. 檢測流程

```javascript
async function analyzeMessage(message) {
  let riskScore = 0;
  const findings = [];

  // 1. 文字分析
  const urgentResult = checkUrgentKeywords(message);
  if (urgentResult.found) {
    riskScore += urgentResult.score;
    findings.push(urgentResult);
  }

  // 2. 威脅檢測
  const threatResult = checkThreatKeywords(message);
  if (threatResult.found) {
    riskScore += threatResult.score;
    findings.push(threatResult);
  }

  // 3. 連結分析
  const linkResult = detectSuspiciousLinks(message);
  if (linkResult.suspiciousLinks > 0) {
    riskScore += linkResult.suspiciousLinks * 20;
    findings.push({
      type: "suspicious_links",
      found: true,
      score: linkResult.suspiciousLinks * 20,
      details: linkResult,
    });
  }

  // 4. 個資檢測
  const personalInfoResult = checkPersonalInfoKeywords(message);
  if (personalInfoResult.found) {
    riskScore += personalInfoResult.score;
    findings.push(personalInfoResult);
  }

  // 5. 金錢要求檢測
  const moneyResult = checkMoneyRequestKeywords(message);
  if (moneyResult.found) {
    riskScore += moneyResult.score;
    findings.push(moneyResult);
  }

  // 6. 權威冒充檢測
  const authorityResult = checkAuthorityKeywords(message);
  if (authorityResult.found) {
    riskScore += authorityResult.score;
    findings.push(authorityResult);
  }

  // 7. 文法錯誤檢測
  const grammarResult = checkGrammarErrors(message);
  if (grammarResult.errors > 2) {
    riskScore += 10;
    findings.push({
      type: "grammar_errors",
      found: true,
      score: 10,
      details: grammarResult,
    });
  }

  // 8. 語氣強迫檢測
  const toneResult = checkForcefulTone(message);
  if (toneResult.forceful) {
    riskScore += toneResult.score;
    findings.push(toneResult);
  }

  // 9. 風險評估
  const riskLevel = assessRiskLevel(riskScore);

  return {
    riskScore,
    riskLevel,
    findings,
    recommendation: getRecommendation(riskLevel),
  };
}

function assessRiskLevel(score) {
  if (score >= 80) return "極高風險";
  if (score >= 60) return "高風險";
  if (score >= 40) return "中等風險";
  if (score >= 20) return "低風險";
  return "安全";
}

function getRecommendation(level) {
  const recommendations = {
    極高風險: "⚠️ 高度懷疑為詐騙訊息！請勿點擊任何連結，立即刪除並回報。",
    高風險: "⚠️ 可能是詐騙訊息！請仔細確認，勿提供任何個資或進行金錢交易。",
    中等風險: "🔍 需進一步確認！請與官方管道聯繫，勿直接回應。",
    低風險: "⚠️ 請留意！訊息內容可能有疑慮，建議提高警覺。",
    安全: "✅ 訊息內容安全，但仍建議保持警覺。",
  };

  return recommendations[level] || "請小心處理此訊息。";
}
```

### 5. 使用方式


### 8. 整合應用

#### 簡訊 APP 整合

```javascript
// 簡訊接收時自動分析
function onReceiveSms(message) {
  analyzeMessage(message).then((result) => {
    if (result.riskLevel === "極高風險" || result.riskLevel === "高風險") {
      showWarningNotification(result.recommendation);
      blockMessage();
    }
  });
}
```

#### LINE/WhatsApp 整合

```javascript
// 訊息檢測機器人
const bot = new LineBot();
bot.on("message", async (event) => {
  const message = event.message.text;
  const result = await analyzeMessage(message);

  if (result.riskLevel === "極高風險") {
    await bot.reply(event.replyToken, {
      type: "text",
      text: "偵測到可能為詐騙訊息！請小心處理：" + result.recommendation,
    });
  }
});
```

#### 企業級整合

```javascript
// 企業通訊軟體整合
class FraudDetectionPlugin {
  constructor(options) {
    this.threshold = options.threshold || 80;
  }

  async analyzeAndBlock(message) {
    const result = await analyzeMessage(message);

    if (result.riskScore >= this.threshold) {
      await this.blockMessage(message);
      await this.notifyAdmin(message, result);
      return true;
    }
    return false;
  }
}
```

## 安全使用注意事項

1. **隱私保護**：所有分析在本地端進行，不會上傳至雲端
2. **更新規則**：定期更新詐騙手法資料庫
3. **誤判處理**：提供申訴機制處理誤判
4. **合規性**：遵守個資法規與隱私政策

## 技術優勢

1. **多層檢測**：文字、語意、連結、上下文多層分析
2. **即時響應**：毫秒級分析速度
3. **高準確度**：結合機器學習與規則引擎
4. **低誤判率**：多重確認機制
5. **可擴展性**：支援自訂規則與模型

## 結論

此詐騙訊息偵測系統提供專業、即時、準確的詐騙防護，可有效保護使用者免於各種詐騙手法侵害。
