## 使用方式

### 啟動控制台

```bash
cd /path/to/opencode
bun run script/security-console.ts
```

### 快速鍵

| 按鍵 | 功能              |
| ---- | ----------------- |
| 1-5  | 啟動/停止單一服務 |
| 6    | 啟動所有服務      |
| 7    | 停止所有服務      |
| 8    | 系統狀態總覽      |
| 9    | 依賴風險分析      |
| 0    | 退出控制台        |
