# 詐騙訊息偵測

自動分析訊息內容，判斷是否為詐騙訊息。

---

## 觸發關鍵字

| 關鍵字   | 動作           |
| -------- | -------------- |
| 分析訊息 | 執行詐騙分析   |
| 檢查詐騙 | 判斷是否為詐騙 |
| 掃描簡訊 | 掃描訊息       |

---

## 風險評分

| 檢測項目 | 分數  | 說明               |
| -------- | ----- | ------------------ |
| 緊急用語 | 20-40 | 緊急、立即、馬上   |
| 威脅語句 | 30-50 | 帳號鎖定、損失財產 |
| 陌生連結 | 40-60 | 不明網址、短網址   |
| 個資索取 | 50-70 | 身分證、银行帳號   |
| 金錢要求 | 60-80 | 匯款、轉帳         |
| 權威冒充 | 30-50 | 銀行客服、警政署   |
| 文法錯誤 | 10-20 | 錯字、語句不通     |
| 語氣強迫 | 15-30 | 情緒勒索           |

---

## 風險等級

| 等級     | 分數  | 建議                   |
| -------- | ----- | ---------------------- |
| 極高風險 | ≥80   | 勿點擊連結，立即刪除   |
| 高風險   | 60-79 | 可能是詐騙，勿提供個資 |
| 中等風險 | 40-59 | 需進一步確認           |
| 低風險   | 20-39 | 請留意，保持警覺       |
| 安全     | <20   | 內容安全               |

---

## 輸出格式

```json
{
  "riskScore": 85,
  "riskLevel": "極高風險",
  "findings": [
    { "type": "urgent", "score": 20 },
    { "type": "threat", "score": 30 },
    { "type": "suspicious_link", "score": 40 }
  ],
  "recommendation": "⚠️ 高度懷疑為詐騙訊息！請勿點擊任何連結"
}
```

---

## 快取

- 相同內容不重複分析（24小時）
- 風險等級快取（7天）
