# 計散機架構 (Ji San Ji Framework)

一個完全獨立的AI系統，擁有自己的模型、知識庫、搜尋引擎和問題解決器，不依賴於任何外部API或廠商模型。

## 架構概述

計散機架構提供了一個完全獨立、智能學習、安全可靠的AI系統。透過模組化的設計、離線運行的能力，以及持續學習的機制，確保系統在各種環境下都能提供穩定和高效的服務。

## 核心模組

### 1. 模型層 (Model Layer)

- 支援本地模型：Ollama、LM Studio、自定義模型
- 模型管理：自動切換、性能監控、信心評估
- 離線運行：即使沒有網路也能正常工作

### 2. 知識庫 (Knowledge Base)

- 技術知識：程式語言、框架、工具、資料庫
- 專案知識：當前專案和歷史專案資訊
- 問題解決知識：常見錯誤和解決方案
- 最佳實踐知識：開發標準和安全規範

### 3. 搜尋引擎 (Search Engine)

- 程式碼搜尋：本地索引的程式碼片段
- 文檔搜尋：技術文檔和API參考
- 網路搜尋：快取的網路資源
- 多策略搜尋：自動選擇最佳搜尋策略

### 4. 問題解決器 (Problem Solver)

- 多層級分類：從構建錯誤到執行錯誤的完整分類
- 智能策略：自動選擇最佳解決方案
- 解決歷史：保存所有解決方案供未來參考
- 模式識別：基於歷史數據的問題識別

### 5. 記憶系統 (AIMemory)

- 偏好記憶：用戶的程式碼風格和回應偏好
- 事實記憶：重要的專案資訊和技術事實
- 指令記憶：用戶的核心指令和禁止事項
- 上下文記憶：專案的當前狀態和上下文

### 6. 整合處理器 (JiSanJiProcessor)

- 核心處理流程
- 安全性檢查
- 問題識別
- 知識檢索
- 模型選擇
- 搜尋補充
- 解決方案生成
- 記憶更新
- 監控記錄

### 7. 監控系統 (SelfMonitor)

- 健康檢查：定期檢查系統健康狀態
- 效能監控：實時追蹤系統效能指標
- 警報系統：自動發送異常警報
- 資源監控：CPU、記憶體、磁碟空間監控

### 8. 安全防護 (SecurityShield)

- 資料保護：自動檢測和過濾敏感資訊
- 權限管理：細粒度的權限控制
- 離線安全：離線模式下仍保持安全標準
- 威脅檢測：識別和阻止危險操作

### 9. 學習系統 (ContinuousLearning)

- 持續學習：每一次互動都會記錄和分析
- 個人化模型：根據用戶偏好調整回應風格
- 知識擴展：自動發現和整合新知識
- 模型微調：基於用戶資料進行模型優化

### 10. 個人化模型 (PersonalModel)

- 程式碼風格偏好
- 回應格式偏好
- 工具偏好
- 信心度評估
- 學習進度追蹤

### 11. 離線模式 (OfflineMode)

- 離線檢查
- 離線行為
- 離線搜尋
- 本地模型管理

## 安裝和使用

### 安裝

```bash
# 進入工作目錄
cd packages/opencode

# 安裝依賴
bun install

# 構建和測試
bun run ji-san-ji-build.sh
```

### 基本使用

```javascript
// 建立系統
const jiSanJi = new JiSanJiSystemImpl()

// 處理問題
const result = await jiSanJi.process("為什麼我的TypeScript構建失敗？", {
  priority: "high",
  context: {
    project: {
      name: "opencode",
      type: "IDE",
      stack: ["TypeScript", "Bun", "SolidJS"],
    },
    user: {
      preferences: {
        language: "zh-TW",
        detailLevel: "medium",
      },
    },
  },
})

console.log(result)
/*
{
  success: true,
  solution: "更新依賴並修復類型定義",
  confidence: 0.92,
  steps: ["檢查package.json", "執行bun install", "修復tsconfig.json"],
  knowledgeUsed: ["typescript", "build", "dependencies"],
  timeTaken: 120,
  modelUsed: "local-llama",
}
*/
```

### 離線使用

```javascript
// 檢查是否離線
const isOffline = jiSanJi.isOffline()

if (isOffline) {
  console.log("離線模式已啟動")

  // 使用離線功能
  const offlineResult = await jiSanJi.process("如何修復這個JavaScript錯誤？", { offline: true })
}
```

### 健康檢查

```javascript
// 執行健康檢查
const health = await jiSanJi.healthCheck()

if (!health.healthy) {
  console.log("系統需要維護")
  console.log(health.details)

  // 發送警報
  await jiSanJi.getMonitoringSystem().alertSystem.sendAlert({
    type: "system-health",
    severity: "critical",
    message: "系統健康檢查失敗",
    details: health.details,
  })
}
```

### 個人化設定

```javascript
// 取得個人化模型
const personalModel = await jiSanJi.getPersonalModel()
```
