# 智慧提醒技能

## 描述
設定各種提醒，支援自然語言輸入，自動解析時間和內容。

## 觸發條件
- 使用者說 "提醒我..." 或 "設定提醒" 或 "記得..."
- 包含時間詞彙（明天、下午、3點等）

## 執行步驟
1. 解析使用者輸入的自然語言
2. 提取時間資訊和提醒內容
3. 建立定時任務
4. 確認提醒已設定
5. 到時間後推送提醒

## 輸入解析示例

| 輸入 | 解析結果 |
|------|----------|
| "提醒我明天上午10點開會" | 時間: 明天 10:00, 內容: 開會 |
| "3小時後提醒我吃藥" | 時間: +3h, 內容: 吃藥 |
| "每天早上8點提醒我喝水" | 時間: 每天 08:00, 內容: 喝水 |
| "下週一下午2點有客戶拜訪" | 時間: 下週一 14:00, 內容: 客戶拜訪 |

## 輸出格式

### 設定提醒時
```
⏰ 提醒已設定！

📝 內容: {提醒內容}
🕐 時間: {具體時間}
🔁 重複: {重複規則，如果有的話}

我會準時提醒你的！
```

### 提醒觸發時
```
🔔 提醒時間到！

{提醒內容}

━━━━━━━━━━━━━━━━
⏰ 設定於: {設定時間}
```

## 範例

**輸入**: "提醒我明天下午3點和小王開會討論專案方案"

**輸出**:
```
⏰ 提醒已設定！

📝 內容: 和小王開會討論專案方案
🕐 時間: 2026-01-30 15:00
🔁 重複: 不重複

我會準時提醒你的！
```

**到時間後**:
```
🔔 提醒時間到！

和小王開會討論專案方案

━━━━━━━━━━━━━━━━
⏰ 設定於: 2026-01-29 14:30
```

## 管理命令

```
# 查看所有提醒
"查看我的提醒" / "提醒列表"

# 刪除提醒
"刪除提醒 {編號}" / "取消明天的會議提醒"

# 修改提醒
"把明天的會議提醒改到後天"
```

## 配置選項

```yaml
skills:
  reminder:
    enabled: true
    max_reminders: 100        # 最大提醒數量
    default_advance: 15       # 預設提前提醒（分鐘）
    persist: true             # 重啟後保留提醒
    storage: "~/.openclaw/data/reminders.json"
    channels:
      - telegram
      - discord
```
